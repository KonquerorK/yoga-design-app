# yoga

Simple yoga design app with flutter (with theme mode available).

## Getting Started

Clone this repo 

```bash
git clone https://github.com/KonquerorK/yoga-design-app.git
```
and
```bash
cd yoga-design-app
```
```bash
flutter pub get
```
```bash
flutter run
```
